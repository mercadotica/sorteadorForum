❓ Como criar a chave da API do YouTube (uma vez só)
  1. Acesse console.cloud.google.com e faça login.
  2. Crie um projeto (ou use um existente).
  3. No menu, vá em APIs e serviços → Biblioteca, busque YouTube Data API v3 e clique em Ativar.
  4. Vá em APIs e serviços → Credenciais → Criar credenciais → Chave de API.
  5. Copie a chave (começa com AIza...) e cole no campo acima.

A chave é gratuita. Cada dia há uma cota de uso de sobra para vários sorteios. A live precisa estar ao vivo com chat aberto no momento da captura.
